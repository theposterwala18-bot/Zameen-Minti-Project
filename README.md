# Zameen Minti Calculator / ਜ਼ਮੀਨ ਮਿਣਤੀ ਕੈਲਕੂਲੇਟਰ

Punjabi/English land measurement calculator for Punjab style units.

## Correct Punjabi Terms Used
- ਮਿਣਤੀ
- ਕਨਾਲ / ਕਨਾਲਾਂ
- ਏਕੜ/ਕਿੱਲਾ
- ਮਰਲਾ / ਮਰਲੇ
- ਵਿਸ਼ਵਾਸੀਆਂ
- ਬਿੱਘਾ

## How to run
1. Unzip folder
2. Open `index.html` in Chrome
3. Use calculator

## Owner Settings
Default password: `••••••`

You can change unit values from Owner Settings. Values are saved in browser localStorage.

## Included
- Rectangle/Square calculator
- Unequal 4-side plot calculator
- Triangle plot calculator
- Punjabi/English language toggle
- Permanent unit reference panel
- Owner protected formula settings
- Print report
