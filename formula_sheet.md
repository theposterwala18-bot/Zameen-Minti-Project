## Rectangle / Square
Area = Length × Width

## Unequal 4-Side Plot
Average Width = (Top + Bottom) / 2  
Average Length = (Left + Right) / 2  
Area = Average Width × Average Length

## Triangle
S = (A + B + C) / 2  
Area = √ S(S-A)(S-B)(S-C)
